# Projeto de criação de um NOTEBOOKLM 

OBJETIVO
--------

O objetivo é utilizar a ferramenta de IA que auxiliar na criação de resumo, apresentações, relatóios , além de criar um chatbot que responde perguntas com base 
nas fontes inseridas pelo usuário e também possibiita a pesquisa pela ferramenta de outras fontes de dados. Nesse repositorio foi anexado arquivos de imagem do 
Noteooklm, do Mapa mental feito pela ferramenta e apresentação gerada em formato pdf.
